# MVP2.1-005 — UI

## Goal

Переработать интерфейс тренировок: **mobile-first**, три режима ввода плана, превью парсинга, удобный отчёт со вставкой текста.

## Requirements

### Общий layout

- Breakpoints: mobile `< 768px`, desktop `≥ 768px`
- Неделя на desktop: **7 колонок** (как таблица тренера) или горизонтальный скролл
- Неделя на mobile: **вертикальный список карточек** по дням
- Sticky header с навигацией по неделе (`PeriodNavigator` — доработать)

### Тренер — создание плана

**Переключатель режима** на странице `/coach/athletes/:id/training/create`:

| Режим       | UI                                                |
| ----------- | ------------------------------------------------- |
| Структурный | текущий `TrainingPlanEditor` (MVP 2)              |
| Один день   | одна большая `textarea` + дата                    |
| Неделя      | одна `textarea` (вставка из Excel) + `start_date` |

**Flow текстового режима:**

1. Ввод текста
2. Кнопка «Разобрать» → `POST .../parse`
3. Превью: 7 карточек / collapsible с warnings
4. Опционально: переключиться в структурный редактор для правки
5. «Сохранить» → `POST .../from-text`

Подсказка-плейсхолдер в textarea — сокращённый пример дня из product brief.

### Тренер — просмотр

- Таб «План» / «Как текст» — переключение structured vs text export
- Карточка дня: тип (иконка run/bike/gym), компактный список блоков
- Итоги спортсмена: ссылка «Garmin» если есть URL

### Спортсмен — отчёт

**Новый flow `WorkoutReportForm`:**

1. Поле **«Вставьте ответ как в мессенджере»** (textarea, крупное на mobile)
2. Кнопка «Разобрать» — клиентский вызов parse API или локальный preview endpoint
3. Автозаполнение: garmin link preview, suggested difficulty
4. Слайдеры трудности / эмоций (можно подправить)
5. Отправка

Для дня без Garmin — только «нагрузка 6-7» достаточно после разбора.

### Спортсмен — просмотр плана

- Карточки дней с читаемым текстом (не только nested cycles UI)
- Свайп / стрелки между днями на mobile

### Доступность и UX

- Touch targets ≥ 44px
- Ошибки парсинга — список warnings, не modal alert
- Loading skeletons для недели
- `npm run lint` и `npm run format:check`

### Не входит

- Кнопка «Создать с ИИ» (MVP 2.2)
- Чат-интерфейс
