## Зависимости

- MVP 2.2 полностью закрыт: удобный ввод, ИИ-ассистент планов и отчётов
- В MVP 1 уже есть ручные ссылки на Garmin Connect — MVP 3 заменяет ручной ввод автоматическим импортом

## Критерии готовности

- Спортсмен авторизует доступ к Garmin API
- Активности импортируются автоматически без ручного копирования ссылок
- График успехов строится на данных, полученных с устройства
- Сбои интеграции обрабатываются явно; пользователь видит статус синхронизации
