# MVP3-005 — UI

## Goal

Собрать интерфейс MVP 3: подключение Garmin, статус синхронизации, список активностей и график успехов на данных с устройства.

## Requirements

### Кабинет спортсмена

- Кнопка «Подключить Garmin» → OAuth flow
- Индикатор статуса подключения и последней синхронизации
- Кнопка «Синхронизировать сейчас» (ручной запуск)
- Список импортированных активностей с фильтрами
- График успехов на данных с устройства

### Кабинет тренера

- Статус подключения Garmin у каждого спортсмена
- Просмотр активностей и графика успехов выбранного спортсмена

### Общее

- Состояния: не подключено, синхронизация, ошибка, требуется переподключение
- Ручные ссылки Garmin из MVP 1 остаются доступны рядом с импортированными данными
- `npm run lint` и `npm run format:check` проходят

## Acceptance Criteria

- [ ] Спортсмен подключает Garmin и видит статус в UI
- [ ] Список активностей отображается с фильтрами
- [ ] График успехов строится на импортированных данных
- [ ] Тренер видит активности и график своего спортсмена
- [ ] Ошибки синхронизации отображаются понятно
- [ ] Основные сценарии покрыты UI/integration-тестами (если инфраструктура готова)

## Dependencies

- [MVP3-004-search.md](MVP3-004-search.md) закрыт
- Backend API MVP 3 полностью работает
- Кабинеты и аналитика из MVP 0–2 существуют

## Декомпозиция

```
MVP3-005 — UI
 ├── подключение Garmin (OAuth)
 ├── статус синхронизации и список активностей
 └── график успехов (данные с устройства)
```
