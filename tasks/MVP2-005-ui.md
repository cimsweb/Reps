# MVP2-005 — UI

## Goal

Собрать интерфейс MVP 2: ИИ-ассистент для программ и отчётов, таблица результатов и график динамики показателей.

## Requirements

### Кабинет тренера

- Форма параметров программы + кнопка «Сгенерировать с ИИ»
- Редактор черновика программы; кнопки «Подтвердить» / «Отменить»
- Выбор спортсмена для просмотра аналитики
- Таблица результатов и график динамики выбранного спортсмена

### Кабинет спортсмена

- Форма кратких заметок + кнопка «Расширить с ИИ»
- Редактор черновика отчёта; кнопки «Подтвердить» / «Отменить»
- Своя таблица результатов и график динамики

### Общее

- Индикатор загрузки при ожидании ответа ИИ
- Ошибка ИИ отображается без технических деталей
- Черновик явно отличается от подтверждённой записи
- `npm run lint` и `npm run format:check` проходят

## Acceptance Criteria

- [ ] Тренер генерирует и подтверждает программу через UI
- [ ] Спортсмен генерирует и подтверждает отчёт через UI
- [ ] Таблица результатов отображается в UI
- [ ] График динамики интерактивен (выбор метрики и периода)
- [ ] Состояние загрузки ИИ обработано
- [ ] Основные сценарии покрыты UI/integration-тестами (если инфраструктура готова)

## Dependencies

- [MVP2-004-search.md](MVP2-004-search.md) закрыт
- Backend API MVP 2 полностью работает
- Кабинеты по ролям из MVP 0 существуют

```

 ├── ИИ-ассистент тренера (программа)
 ├── ИИ-ассистент спортсмена (отчёт)
 └── таблица результатов + график динамики
```
