# MVP1-004 — Search

## Goal

Дать тренеру и спортсмену возможность находить и просматривать релевантные данные: список спортсменов, профили, рекорды, историю обратной связи и ссылки на Garmin.

> В контексте Reps «поиск» — это запросы и фильтрация данных, а не полнотекстовый или семантический поиск.

## Requirements

### Для тренера

- Список приглашённых и принятых спортсменов
- Статус приглашения: ожидает / принято / отклонено
- Просмотр профиля выбранного спортсмена
- Просмотр личных рекордов спортсмена
- Просмотр истории обратной связи по тренировкам
- Просмотр прикреплённых ссылок Garmin Connect

### Для спортсмена

- Список своих тренеров (принятые связи)
- Просмотр и редактирование своего профиля
- Просмотр своих рекордов
- Просмотр своей истории обратной связи

### Доступ и безопасность

- Тренер видит только своих спортсменов
- Спортсмен видит только свои данные
- Неавторизованный и чужой доступ возвращает 403/404 без утечки данных
- Пагинация для списков feedback и рекордов (если > 20 записей)

## Acceptance Criteria

- [x] Тренер видит список своих спортсменов с корректными статусами
- [x] Тренер открывает профиль, рекорды и feedback конкретного спортсмена
- [x] Спортсмен видит список своих тренеров
- [x] Спортсмен видит только свои профиль, рекорды и feedback
- [x] Попытка доступа к чужим данным блокируется
- [x] API-ответы соответствуют задокументированному контракту (MVP1-001)
- [x] Unit-тесты на правила доступа
- [x] Integration-тесты на типовые сценарии просмотра

## Dependencies

- Связи и агрегация данных работают
