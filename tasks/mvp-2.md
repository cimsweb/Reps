# MVP 2 — Планы тренировок и отчёты

**Этап:** Рост  
**Предшественник:** [mvp-1.md](mvp-1.md)  
**Следующий:** [mvp-2.1.md](mvp-2.1.md)

## Цель

Тренер составляет план тренировок для спортсмена, спортсмен выполняет и фиксирует результат — всё в Reps, без таблиц и мессенджеров.

## Scope

### Для тренера

- [x] Создание плана тренировки для спортсмена: на один день и на неделю (формы, **без ИИ**)
- [x] Типы тренировок: бег, велосипед, зал
- [x] Структура занятия: набор упражнений, несколько циклов, списки подходов/интервалов
- [x] Просмотр плана на неделю и на месяц вперёд
- [x] Просмотр результатов тренировок спортсмена за неделю и за месяц

### Для спортсмена

- [x] Отчёт по итогу тренировки: оценка трудности 0–10 (10 — тяжёлая нагрузка)
- [x] Отчёт по итогу тренировки: эмоциональное состояние 0–10 (10 — «кайфую»)
- [x] Просмотр плана тренировок на неделю и на месяц
- [x] Просмотр своих итогов тренировок за неделю и за месяц

## Вне scope

- ИИ для генерации программ и расширения отчётов (отдельный этап после MVP 2)
- Таблица результатов по упражнениям и дистанциям, графики динамики показателей
- Автоматический импорт активностей с устройств (Garmin API — MVP 3)
- Автоматические алерты о перегрузке и отставании от плана
- Планирование дальше чем на видимый горизонт (месяц) в UI календаря

## Зависимости

- MVP 1 полностью закрыт: связка тренер ↔ спортсмен, профили, рекорды, обратная связь

## Критерии готовности

- Тренер вручную создаёт план на день и на неделю для связанного спортсмена
- План поддерживает бег, велосипед и зал с циклами и списками упражнений
- Тренер и спортсмен видят план на неделю и на месяц вперёд
- Спортсмен оставляет отчёт с оценкой трудности и эмоций (0–10)
- Тренер и спортсмен видят итоги тренировок за неделю и за месяц
- Доступ к планам и отчётам только в рамках связи coach ↔ athlete
