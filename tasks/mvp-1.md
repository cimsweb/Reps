# MVP 1 — Связка тренер ↔ спортсмен

**Этап:** После курса  
**Предшественник:** [mvp-0.md](mvp-0.md)  
**Следующий:** [mvp-2.md](mvp-2.md)

## Цель

Тренер и спортсмен работают через Reps без сторонних инструментов.

## Scope

- [x] Тренер приглашает спортсмена по email
- [x] Спортсмен оставляет обратную связь по итогу тренировки
- [x] Ссылки на отчёты в Garmin Connect
- [x] Профиль спортсмена: рост, вес, возраст, пол
- [x] Личные рекорды на дистанциях и в упражнениях

## Вне scope

- Автоматический импорт данных из Garmin (только ссылки вручную)
- ИИ для составления программ и отчётов
- Таблицы результатов и графики динамики показателей
- Автоматическое выявление перегрузки и стагнации

## Зависимости

- MVP 0 полностью закрыт: роли тренер и спортсмен работают, аутентификация стабильна

## Критерии готовности

- Тренер может пригласить спортсмена; спортсмен принимает приглашение
- Спортсмен заполняет профиль и фиксирует рекорды
- Спортсмен оставляет обратную связь после тренировки
- Можно прикрепить ссылку на отчёт в Garmin Connect
- Тренер видит своих спортсменов и их активность в рамках MVP 1
