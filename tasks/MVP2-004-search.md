# MVP2-004 — Search

## Goal

Дать тренеру и спортсмену доступ к аналитике: таблица результатов по упражнениям и дистанциям, график динамики показателей, история программ и отчётов.

> В контексте Reps «поиск» — запросы, фильтрация и выборка данных для таблиц и графиков.

## Requirements

### Таблица результатов

- Строки: упражнение или дистанция
- Колонки: последнее значение, лучшее значение, дата
- Фильтр по типу: дистанция / упражнение
- Только данные выбранного спортсмена

### График динамики

- Ось X — дата; ось Y — значение показателя
- Выбор метрики из доступных для спортсмена
- Период: последние 30 / 90 / 365 дней (или весь период)

### Программы и отчёты

- Тренер видит список своих программ (черновики и подтверждённые)
- Спортсмен видит свои отчёты

### Безопасность

- Тренер видит аналитику только своих спортсменов
- Спортсмен видит только свою аналитику
- Чужие данные — 403/404

## Acceptance Criteria

- [ ] Таблица результатов отображает корректные данные по упражнениям и дистанциям
- [ ] График динамики строится для выбранной метрики и периода
- [ ] Тренер фильтрует аналитику по своему спортсмену
- [ ] Спортсмен видит только свои данные
- [ ] API-ответы соответствуют контракту (MVP2-001)
- [ ] Unit-тесты на правила доступа
- [ ] Integration-тесты на типовые запросы аналитики
